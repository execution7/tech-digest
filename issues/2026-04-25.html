🔥 头条速递

今天最让人意外的新闻：Firefox 149 悄悄内置了 Brave 的 adblock-rust 广告拦截引擎。

没有发布会，甚至连发行说明里都没有高调提及。Mozilla 直接把 Brave VP 开发的 Rust 广告拦截核心塞进了 Firefox，默认关闭，通过 about:config 手动启用后可以加载 EasyList 和 EasyPrivacy 规则集。

这事为什么值得当头条？三个原因：

1. 竞争对手共享核心代码极为罕见。 浏览器厂商之间通常在底层引擎上各自为战，Firefox 直接拿 Brave 的代码来用，说明在广告拦截这件事上，"用户利益" 赢过了 "NIH 综合征"。

2. Rust 生态的跨项目复用正在兑现承诺。 adblock-rust 之所以能被直接集成，是因为它是一个设计干净、接口清晰的 Rust crate。这不是第一次了 —— 想想 Firefox 的 Stylo（从 Servo 移植的 CSS 引擎）。Rust 正在成为浏览器底层的通用语言。

3. 广告拦截正在从"插件"升级为"浏览器内核功能"。 当 Chrome 用 Manifest V3 削弱扩展能力的时候，Firefox 选择把广告拦截直接做进引擎层。这是一个很明确的战略分化信号。

判断： 虽然现在默认关闭，但方向已经很清楚。对于关心隐私的用户来说，Firefox 作为"非 Chrome 浏览器"的独特价值正在持续增强。
